# Zero Eisenhower Matrix

## Urgent And Important

Tasks that generate sales, deliver work, or unblock a real client.

Examples:

- Prepare a meeting with a key client.
- Prepare a pitch for a hot vertical.
- Fix the main product landing page.
- Create sample report.
- Respond to hot lead.

## Important Not Urgent

Assets that increase future capacity.

Examples:

- Pattern library.
- Zero framework.
- Prompt library.
- Case studies.
- Tool testing.
- Report automation.

## Urgent Not Important

Tasks that feel pressing but do not change the business.

Examples:

- Minor visual changes.
- Repeated logo changes.
- Reviewing new tools without applying them.
- Perfecting copy before talking to customers.

## Neither Urgent Nor Important

Distractions.

Examples:

- New unrelated ideas.
- Canceled tools.
- Exploring technology without a use case.
