# Zero Operating System

The source of truth for how Zero evaluates, researches, sells, builds, and improves projects. It is a venture-operating framework: a way to turn any idea or client into a repeatable pipeline that compounds into reusable assets.

## Read in this order

1. `zero-venture-framework.md` — the core philosophy and pipeline
2. `decision-rules.md` — what to accept, prioritize, and avoid
3. `eisenhower-matrix.md` — how to triage day-to-day work

The companion `../scripts/` and `../agent-os/` folders are the tooling that puts this framework into practice: a single-Worker control plane plus CLI helpers for a kanban / investigation / approval loop.
